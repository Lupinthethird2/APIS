1. Introducción

En este proyecto, utilizaremos 2 APIS para obtener datos sobre usuarios y publicaciones, en el caso de la primera, mientras que en el otro ubicaremos los eventos deportivos, los agruparemos por deporte y fechas para luego mostrarlo y graficarlo. Vamos a transformar los datos en información estructurada, aplicar estadísticas descriptivas y crear visualizaciones usando Python y bibliotecas como requests, pandas, matplotlib, y seaborn.

2. Objetivos:
   Generar tablas estadisticas y graficos de los datos obtenidos.
   Ejecutar de manera correcta los codigos que nos permiten acceder a las APIS, asi como transformarla a python usando JSON
   Visualizar la flexibilidad que nos ofrecen las APIS usando la funcion get.
   
3. Desarrollo y conclusion:

Este análisis ha proporcionado una visión general de los usuarios y sus publicaciones utilizando la API de JSONPlaceholder.Asi como una vision general a los eventos deportivos disponibles usando la API de TheSportsDB. Hemos explorado estadísticas descriptivas y visualizaciones sobre las ciudades de los usuarios y el número de publicaciones por usuario.
